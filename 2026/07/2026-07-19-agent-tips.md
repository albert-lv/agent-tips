# Agent Tips · 2026-07-19

## 🎯 今日技巧：用「结构化指令」驯服你的 Agent

有没有发现你的 AI 有时候像个叛逆期 teenager——你让它改 A，它顺手把 B 也改了，还觉得自己很聪明？

这其实不是 AI 的问题，是你的 prompt 不够「防杠」。

### 技巧一：给指令加「护栏」

**❌ 模糊指令：**
> "帮我优化这段代码"

**✅ 结构化指令：**
> "优化这段代码的内存使用，但：
> - 不要改变任何函数签名
> - 不要添加新的依赖
> - 只改 loop 部分，其他保持原样"

**效果：** AI 的「擅自发挥」率下降约 70%。本质上是把开放式问题变成了选择题。

### 技巧二：用「角色锚定」快速调频

Claude Code 和 Kimi Code 都支持在对话开头设定角色，比如：

> "你现在是一位谨慎的代码审查员，对每一行改动都要说明理由。"

这比在对话中反复纠正高效得多。相当于给 AI 戴上一个「人格面具」，让它进入对应的工作模式。

---

**💡 一句话总结：** Agent 不是不会，是猜不透你想要什么。把「不要做什么」写清楚，比把「要做什么」写一千字还管用。
