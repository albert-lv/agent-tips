#  Claude Code 的斜杠命令：你的「后悔药」和「压缩包」

> **日期**：2026-06-14 | **第 6 弹**
> 
> 今天聊点 Claude Code 里藏得挺深的实用命令 —— 不是那种需要你背半天的快捷键，而是几个打 `/` 就能召唤出来的「后悔药」和「整理术」。

---

##  `/compact` —— 话太多？一键压缩

跟 Claude Code 聊久了，上下文会越来越长，它开始「健忘」或者回复变慢。这时候别急着新开窗口，直接输：

```
/compact
```

Claude 会**自动总结你们刚才聊的所有内容**，把长篇大论压缩成一段摘要，然后清掉冗余历史。对话不会断，但它「失忆」了只记得精华，继续聊很清爽。

**什么时候用**：
- 聊了十几轮，感觉它开始重复问同一个问题
- 要切换子任务（比如从「改 bug」切换到「写测试」），但还想保留核心上下文
- 上下文快满了（Claude 会提示你）

> 💡 **Pro tip**：`/compact` 之后，你可以立刻补一句「刚才的总结别忘了，现在我们做 X」。丝滑衔接，不用从头解释。

---

##  `/undo` —— 改崩了？秒回退

Claude Code 改代码之前会自动创建一个 git checkpoint，但有时候你连着让它改了三四处，突然发现「第二步就错了」。这时候：

```
/undo
```

直接回退**上一次的编辑操作**，不需要你自己 `git checkout` 或者翻历史。可以连续 `/undo` 多次，像 Ctrl+Z 一样。

**什么时候用**：
- 「你先这样改…… 等等，不对，回退一下」
- 批量重构后，发现某个文件的改动不是想要的
- 想快速对比「改前 vs 改后」

> ⚠️ 注意：`/undo` 只回退 Claude 做的编辑，你自己手改的代码不受影响。

---

##  bonus：`/cost` —— 看看今天「烧」了多少

好奇今天跟 Claude 聊了半天花了多少 token？

```
/cost
```

直接输出当前会话的 API 调用次数和估算费用。适合在「免费额度要没了」或者「老板问我这个月为什么用了这么多」的时候自查 😂

---

## 今日一句话

> **「/compact 整理脑容量，/undo 保住发际线。」**
> 
> 两个命令都不难记，但用熟了会大幅改善你跟 Claude Code 的聊天体验 —— 不再怕聊太长，也不怕手滑改错。
